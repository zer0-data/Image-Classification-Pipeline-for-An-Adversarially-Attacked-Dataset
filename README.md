# Image Classification Pipeline for An Adversarially Attacked Dataset

The Repo contains two distinct image classification pipelines which were submitted and tested in the **Pixelplay 2025** Competition, The folder names are the accuracies the methods achieved and reflected on the final leaderboard  

* **Animal Classification with Continuous Attribute Learning (CAL)**
* **CLIP-based Image Classification Using Multiple Models**

The second methodology aided me in my 2nd place finish in the hackathon, it was robust system which could deal with the adversarial attacks and unseen classes in the datatset.
